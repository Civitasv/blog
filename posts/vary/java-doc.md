`javadoc`是 JDK 中很实用的工具，它可以由源文件生成一个 HTML 文档。

常用命令总结如下：

`@param`: 方法参数注释

`@return`: 方法返回值注释

`@throws`: 方法抛出异常注释

`@since`: 起始版本注释

`@author`: 作者注释

`@version`: 版本注释

`@see`

`@link`
